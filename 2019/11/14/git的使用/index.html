github本地项目的初始配置

cmd 切换到指定路径
配置 git config --global user.name '账户名'
配置 git config --global user.email 'b347996447@gmail.com'    
检查配置 git config -l

本地关联到github项目

1.当前目录下打  git init
2.项目关联github git remote add origin + git@github.com:Yangzhenzhong1/blockchain.git(仓库链接)

如何把本地的文件推送到git端

1.git status 看本地发生哪些变化
2.如何让文件修改在本地生效
    1.git add *
    2.git commit -m '注释'
3.推送到git端
    1.git push -u origin master

如何获得别人git项目

1.搜索项目
2.fork项目

自己如何查看别人的修改并同意修改

fork项目到本地后
1.修改后点 commit changes
2.项目pull request
3.点击new pull request
4.点击 create pull request
5.pull request
6.点击修改内容查看
7.merge pull request  confirm merge 合并

hexo 使用

hexo s 开启博客
hexo new title 创建博客
hexo clean 清理重复文件
hexo d -g 同步到git端.
